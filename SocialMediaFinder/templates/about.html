What is Sherlock Finds?

Sherlock Finds is a powerful web-based tool that searches for usernames across 80+ social media platforms and online services. It's designed to help investigators, researchers, and individuals check username availability or find existing profiles.

Features

  • Search across 80+ popular platforms
  • Real-time results with visual indicators
  • Mobile-responsive web interface
  • Fast and reliable checking
  • No data storage - your searches are private

Supported Platforms

Instagram, Twitter, Facebook, GitHub, LinkedIn, YouTube, Reddit, TikTok, Snapchat, Pinterest, Medium, Steam, Spotify, and many more!

About the Creator

Aditya Deore - A passionate developer who created this enhanced version of the popular Sherlock tool. This project demonstrates expertise in Python, Flask, web development, and API integration.

Technology

Built with Python Flask, modern HTML/CSS/JavaScript, and deployed using cloud hosting services. The tool uses HTTP requests to check username availability across different platforms.

Privacy

This tool doesn't store any search data or usernames. All searches are processed in real-time and results are only displayed to you.
